## Usage

- **Left Arrow / Swipe Left**: Nope (dislike)
- **Right Arrow / Swipe Right**: Like
- **Up Arrow / Swipe Up**: Pass (skip without voting)
- **Down Arrow / Swipe Down**: Hide (permanently remove from rotation)
- **S**: Show results (after 10+ swipes)
